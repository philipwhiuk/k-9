K2 Mail is an open source email client with support for multiple accounts, search, IMAP push email, multi-folder sync, flagging, filing, signatures, BCC-self, PGP/MIME & more! 

K2 supports IMAP, POP3 and Exchange 2003/2007 (with WebDAV).

Install the app "OpenKeychain: Easy PGP" to encrypt/decrypt your emails using OpenPGP.

K2 is a community developed project.  If you're interested in helping to make the most popular open source email client on Android even better, please join us!  You can find our bug tracker and source code at https://github.com/philipwhiuk/k2
We're always happy to welcome new developers, designers, documenters, bug triagers and friends. 

If you're having trouble with K2, please report a bug at https://github.com/philipwhiuk/k2 rather than just leaving a one-star review. We don't mind you telling the world that you're frustrated, but if you use our bug tracker, we have a better chance of fixing whatever's giving you a hard time.

You can find K2's release notes at: https://philipwhiuk.github.io/k2/releaseNotes.html
